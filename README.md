# VALKNUTPOWER - Elite Training System

Sistema de entrenamiento avanzado basado en los protocolos de Pedro Colina y Jeremy Ethier.

## Características
- **Modo Pro (Valknut Pro):** Basado en la metodología de Jeremy Ethier.
- **Modo Hybrid:** Sincronizado para entrenamiento en pareja (Pedro & Valentina).
- **Legibilidad:** Optimizado para lectura en el gimnasio con fuentes XL.
- **PWA Ready:** Instalable en dispositivos móviles y funciona offline.

## Instalación
1. Clona este repositorio.
2. Abre `index.html` en tu navegador o despliégalo con GitHub Pages.
3. En tu móvil, selecciona "Añadir a la pantalla de inicio".
